# dotfiles

My personal configurations for GNU/Linux systems using [stow](https://www.gnu.org/software/stow/).

## Excluded

For security reasons, excluded from this repo are:

- SSH configurations
- mbsync config

## Recommended tools

- `i3wm` with `lightdm`.
- `fish` shell with `omf` and `bass`.
- `dmenu`, `i3blocks` and `i3lock-fancy`.
- `scrot` and `xclip`.
- `xautolock`.
- `mate-power-manager`.
- `emacs`
- `feh`
- `blueman-applet`
- `connman-gtk`
- `nvm` and `rvm`
- `git`
- `light`

## Requirements

- GNU Make
- GNU Stow
